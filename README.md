## About

Larisin AI Backend provides two core AI features for UMKM (small businesses) in Indonesia:

- **AI Image Generation** — Upload a product photo and get back a professional, studio-quality version generated by Azure OpenAI's image model. The original and generated images are stored in Azure Blob Storage and returned as public URLs.
- **AI Caption Generation** — Generate 3 social media caption variations with relevant hashtags, tailored to the business type, target audience, and promotion style. Results are persisted in Azure Cosmos DB.

## Tech Stack

| Layer | Technology |
|---|---|
| Language | Python 3.11 |
| Package Manager | [uv](https://docs.astral.sh/uv/) |
| Framework | FastAPI + Uvicorn |
| AI — Text | Azure OpenAI (`gpt-4o`) |
| AI — Image | Azure OpenAI (`gpt-image-1.5`) |
| Storage | Azure Blob Storage |
| Database | Azure Cosmos DB |
| Containerization | Docker |
| CI/CD | GitHub Actions → Azure Container Registry → Azure App Service |

## API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| `GET` | `/` | Health check |
| `POST` | `/api/v1/ai/generate-caption` | Generate captions & hashtags (JSON body) |
| `POST` | `/api/v1/ai/generate-image` | Generate AI product image (multipart form) |

Full interactive docs available at `/docs` (Swagger UI).

## Local Development

```bash
# Install dependencies
uv sync

# Create your .env file and fill in your Azure credentials
cp .env.example .env

# Run the dev server
uv run uvicorn main:app --reload
```
